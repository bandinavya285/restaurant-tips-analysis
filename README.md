# 📊 Restaurant Tipping Data Analysis Pipeline

A Python-based data analysis project that ingests, cleans, and analyzes consumer tipping behavior. This project establishes an isolated virtual environment (`venv`) to process data and extract actionable business insights using Pandas.

## 🚀 Key Features
- **Data Ingestion:** Live streaming of restaurant datasets from remote repositories.
- **Data Preprocessing:** Automated data cleaning and handling of missing values using Pandas.
- **Demographic Filtering:** Segmentation of dining patterns based on party sizes.
- **Statistical Aggregation:** Calculated behavioral trends across different days of the week.

## 🛠️ Tech Stack & Environment
- **Language:** Python 3
- **Libraries:** Pandas, NumPy, Jupyter (ipykernel)
- **IDE:** Visual Studio Code
- **Environment Management:** Python Virtual Environments (`venv`)

## 📈 Key Insight Generated
Through data aggregation, the analysis proved that **Sunday** yields the highest average tip amount ($3.25), while **Friday** yields the lowest ($2.73), indicating distinct weekend spending patterns among consumers.
